What are pseudo classes?

A pseudo-class is used to define a special state of an element

For example, pseudo class can be used to:

  • Style an element when an user hovers about it
  • Style visited and unvisited links with different color
  • Style an element when it get focus

Syntax of pseudo-class

        
        selector:pseudo-class {
        property: value;
      }

Usage Examples

         
        a:link {
          color: red;
        }
        
        
        a:visited {
          color: green;
        }
        
        
        a:hover {
          color: pink;
        }
        
        
        a:active {
          color: blue;
        }
