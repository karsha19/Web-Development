CSS

CSS is known as Cascading Style Sheet. It gives appearance to webpage. It is used for styling, formatting webpage and makes webpage visually attractive

Selectors in CSS

Selector is a way to select an element

  1. Simple Selector
    1. Element Selector
    2. Class Selector
    3. ID Selector
  2. Pseudo-class Selector
  3. Multiple Selector

Element Selector/Tag Selector/Type Selector

CSS can select HTML elements by using an elements tag name.Tag name is the word between angle brackets

This is paragraph with p tag


        <p> This is paragraph with p tag </p>

        <style>
        
        p{
            color: blue;
        }

        <style>

Class Selector

CSS is not limited to selecting elements by tag name. HTML elements can have attributes as well for tags. One common attribute is class attribute. It is possible to select element using class attribute.

To select HTML element by its class using CSS we need to prepend (.)before its class name

This is paragraph with class attribute named as para

        <p class="para"> This is paragraph with class attribute named as para </p>

        <style>
           
           .para{
            color: red;
           }

        </style>

ID Selector

id is also one of the attributes available for tag. There will be unique id for a tag. there can be same class for multiple tags

In Css we can use ID attribute for styling by prepending (#)before idname

This is paragraph with id attribute named id_para

        <p id="id_para">This is paragraph with id attribute named id_para</p>

        <style>
           
           #id_para{
            color: brown;
           }

        </style>

Pseudo-Class Selector

A CSS Pseudo class is a keyword added to a selector that specifies a special state of the selected element. For example hover can be used to change a button's color when the user pointer hovers over it

This pseudo can be used with class and id attributes as well

Click here for learning more about special states of element
        <button>Submit</button>

        <style>
           
           button:hover{
            color: pink;
           }

        </style>

Universal Selector

Universal selector provided by CSS helps in choosing any elements within HTML Page.

We have to use (*) symbol for denoting selector as universal selector

        <style>
            *{
                margin: 10px;
            }
        </style>

Nested Selector

Like in HTML where we have elements nested inside another elements the same can be done in css.

Let's say you have a paragraph tag inside your main content and also one in your footer, but you want the footer's font size to be smaller.You can simply target paragraph tags inside main differently than you would paragraph tags inside footer by nesting the paragraph tag inside its parent.

        <style>
            main p{
                font-size: 1rem;
            }
            footer p{
                font-size: 0.75rem;
            }
        </style>

How to add styling to HTML?

Inline CSS

Inline CSS is styling within the opening tag with style attribute

This para is styled using inline CSS

        <p style="color: orange">This para is styled using inline CSS</p>

Internal CSS

Internal CSS is styling of element using style element in head tag

This was styled using internal css with color:red 
        <head>
            <style>
                div{
                    color: red;
                }
            </style>
        </head>

External CSS

When HTML and CSS code are in seperate files, then we need to link CSS file into HTML
We have to link tag in head element for linking External CSS

        <head>
            <link rel="stylesheet" href="cssfile.css">
        </head>

Colors in CSS

Colors in CSS can be specified by following methods

  • Hexadecimal colors
  • RGB colors
  • Predefined/Cross-browser color names

Hexadecimal Colors

Hexadecimal color is specified with #RRGGBB where RR(red), GG(green) and BB(blue) hexadecimal integers specify the components of color. All the values must be between 00 and FF

  • #FF0000: Red
  • #00FF00: Green
  • #0000FF: Blue

RGB Colors

An RGB Color is specified with rgb() function, which has the following syntax rgb(red,green,blue)

  • rgb(0,0,255): Blue
  • rgb(0,255,0): Green
  • rgb(255,0,0): Red
  • rgb(255,255,255): White
  • rgb(0,0,0): Black

Predefined Colors

Around 140 color names are predefined in HTML and CSS color specification
